# Die, Socket, Die
Stress test a websocket.

## Install

```bash
$ npm install -g diesocketdie
```

## Run

```bash
$ diesocketdie --ad 'ws://A_WEBSOCKET' --cc 25
```

```bash
$ diesocketdie --ad 'ws://A_WEBSOCKET' --cc 25 --hb 10000 --pi 10000 -pw ...etc
```

## Help

```bash
$ diesocketdie -h
```
